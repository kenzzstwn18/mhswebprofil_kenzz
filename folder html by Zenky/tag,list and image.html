Jenis-Jenis tag

berikut adalah jenis dan contoh tag yang digunakan di dalam html

  • Tag kalimat
  1. tag (p) = tulisan/kalimat
  2. tag (b) = mempertebal huruf
  3. tag (u) = garis bawah teks
  4. tag (br) = spasi
  5. tag (hr) = garis heading/garis horizontal
  6. tag (em) = cetak miring
  7. tag (strong) = mempertebal/tegas kalimat
  • Tag pendukung
  1. tag (img) = gambar
  2. tag (h1-h6) = bentuk judul
  3. tag (ul,ol)=li , (dl)=dt,dd = untuk membuat list
  4. tag (a) = untuk membuat link
  • Atribut pendukung image pada website
  1. src = untuk file image
  2. alt = untuk menggunakan nama jika semisalnya image eror
  3. widht = untuk mengubah ukuran gambar
  4. height = untuk mengubah tinggi gambar tetapi sebaiknya gunakan salah satu saja antara widht/height
  5. title = untuk memberikan nama pada gambar di website
